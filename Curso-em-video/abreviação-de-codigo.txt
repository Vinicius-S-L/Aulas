┌───────────────────────────────────────────────────────────────────────────────
│ 1. OPERADORES DE ESTRUTURA (HTML)
└───────────────────────────────────────────────────────────────────────────────
Estes são os "motores" para construir a árvore do documento:

  • Filho (>): Aninha um elemento dentro de outro
    → Exemplo: ul>li

  • Irmão (+): Coloca elementos no mesmo nível
    → Exemplo: div+p

  • Subir Nível (^): Sai de um aninhamento para o nível anterior
    → Exemplo: div>p^span

  • Multiplicação (*): Repete o elemento
    → Exemplo: li*5

  • Agrupamento (()): Cria subestruturas complexas
    → Exemplo: div>(header>nav)+footer

┌───────────────────────────────────────────────────────────────────────────────
│ 2. ATRIBUTOS E CONTEÚDO
└───────────────────────────────────────────────────────────────────────────────
Personalize as tags com classes, IDs e textos:

  • ID (#): Adiciona atributo de ID
    → Sintaxe: div#cabecalho
    → Resultado: <div id="cabecalho">

  • Classe (.): Adiciona atributo de classe
    → Sintaxe: div.item
    → Resultado: <div class="item">

  • Atributo Customizado ([]): Define atributos personalizados
    → Sintaxe: img[src="foto.jpg"]

  • Numeração ($): Usado com multiplicação para sequências
    → Sintaxe: item$*3
    → Resultado: item1, item2, item3

  • Texto ({}): Adiciona conteúdo de texto
    → Sintaxe: p{Olá Mundo}
    → Resultado: <p>Olá Mundo</p>

┌───────────────────────────────────────────────────────────────────────────────
│ 3. ABREVIAÇÕES DE CSS
└───────────────────────────────────────────────────────────────────────────────
O Emmet para CSS foca em abreviar propriedades e valores:

  • Posicionamento:
    → pos:a = position: absolute
    → pos:r = position: relative
    → d:f = display: flex

  • Margem e Padding:
    → m10 = margin: 10px
    → p20 = padding: 20px
    → mt10 = margin-top: 10px

  • Dimensões:
    → w100 = width: 100px
    → h50p = height: 50%

  • Cores:
    → c#f = color: #fff
    → bg#0 = background: #000

  • Texto:
    → tac = text-align: center
    → fz20 = font-size: 20px
    → lh1.5 = line-height: 1.5

┌───────────────────────────────────────────────────────────────────────────────
│ 4. BOILERPLATE (ESTRUTURA INICIAL)
└───────────────────────────────────────────────────────────────────────────────

  • ! ou html:5: Gera a estrutura básica completa de um documento HTML5
